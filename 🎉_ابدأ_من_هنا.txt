🎯 **اختبار سريع للأنظمة:**
═══════════════════════════════════════════════════════════════

### 🔐 2FA (الأحدث):
```bash
# 1. سجّل دخول أولاً:
POST /auth/login

# 2. أنشئ 2FA:
POST /2fa/setup
Authorization: Bearer YOUR_TOKEN

# 3. امسح QR Code بـ Google Authenticator

# 4. فعّل 2FA:
POST /2fa/enable
{
  "token": "123456"
}

# ✅ تم! 2FA مفعّل!
```

### ⚡ WebSocket:
```javascript
// في Frontend:
import io from 'socket.io-client';

const socket = io('http://localhost:3000', {
  auth: { token: 'YOUR_JWT_TOKEN' }
});

// استقبال إشعار:
socket.on('notification:new', (data) => {
  console.log('إشعار جديد:', data);
});

// ✅ يعمل!
```

### ☁️ AWS S3:
```bash
POST /upload/image
Authorization: Bearer YOUR_TOKEN
Content-Type: multipart/form-data

file: [اختر صورة]

# الرد:
{
  "original": {
    "key": "uploads/...",
    "url": "https://..."
  },
  "thumbnail": {
    "url": "https://..."
  }
}

# ✅ تم رفع الصورة لـ S3!
```

### إذا كانت أول مرة:

```bash
# 1. تثبيت Dependencies:
npm install

# 2. إعداد Database:
npx prisma migrate dev
npx prisma generate

# 3. (اختياري) تحميل بيانات تجريبية:
npm run seed

# 4. تشغيل:
npm run start:dev

# ✅ جاهز!
```

### ملف .env:
```env
# Database
DATABASE_URL="mysql://root:@localhost:3306/barber_platform"

# JWT
JWT_SECRET="change-this-in-production-to-strong-secret"
JWT_EXPIRES_IN="7d"

# AWS S3 (اختياري - للتخزين السحابي)
AWS_REGION="us-east-1"
AWS_S3_BUCKET="your-bucket"
AWS_ACCESS_KEY_ID="your-key"
AWS_SECRET_ACCESS_KEY="your-secret"

# Email (اختياري)
SMTP_HOST="smtp.gmail.com"
SMTP_PORT=587
SMTP_USER="your@email.com"
SMTP_PASS="your-password"
```
